## Run Locally

Clone the project

```bash
  git clone https://github.com/SciMathist/yt.py.git
```

Go to the project directory

```bash
  cd yt.py
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Run the application

```bash
  python yt.py
```
